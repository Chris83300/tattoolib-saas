📋 2. CONTRÔLEURS

### DepositController
- **payment(BookingRequest)**: GET - Affiche page paiement
  - Auth: auth()->user()->client
  - Validation: client_id === auth()->user()->client->id
  - Status: ['accepted', 'awaiting_deposit']
  - Vérifications: pas déjà payé, deadline non expirée
  - Retour: view('client.deposit-payment') avec bookingRequest, stripeKey
  
- **process(BookingRequest)**: POST - Crée session Stripe
  - Auth: auth()->user()->client
  - Validation: mêmes vérifications que payment()
  - Stripe: Session::create() avec montant, metadata
  - Success: response()->json(['sessionId' => $session->id])
  - Error: response()->json(['error' => 'Erreur de paiement'], 500)
  
- **success(BookingRequest, Request)**: GET - Page succès
  - Vérifie session_id et !deposit_paid_at
  - Stripe: Session::retrieve($sessionId)
  - Si payment_status === 'succeeded': update(['deposit_paid_at' => now(), 'status' => 'deposit_paid'])
  - Retour: view('client.deposit-success')
  
- **cancel(BookingRequest)**: GET - Page annulation
  - Retour: view('client.deposit-cancel')

### ClientController
- **dashboard()**: GET - Dashboard client
  - Auth: auth()->user()->client
  - Requête: BookingRequest::where('client_id', $client->id)->with(['bookable.user', 'conversation'])
  - Stats: calculées depuis collection
  - Retour: view('client.dashboard')
  
- **chat(Conversation)**: GET - Affiche chat
  - Auth: Auth::user()->client
  - Validation: bookingRequest->client_id === $client->id
  - Chat ouvert: ['accepted', 'deposit_paid'] && conversation->status === 'active'
  - ExpiryInfo: expires_at, days_remaining, time_remaining, warning_message
  - Messages: conversation->messages()->with('sender')->get() si chat ouvert
  - Marquage: messages non lus marqués comme lus
  
- **sendMessage(Request, Conversation)**: POST - Envoi message
  - Auth: Auth::user()->client
  - Validation: content nullable|string|max:2000, attachments.* file|mimes
  - Blocage: pièces jointes interdites si !deposit_paid_at
  - Création: Message::create() avec sender_type = 'client'
  - Upload: foreach attachments -> message->addMedia()
  - Redirection: back()

### BookingRequestController
- **accept(Request, BookingRequest)**: POST - Accepte demande
  - Auth: auth()->user()->tattooer
  - Validation: bookable_id === $tattooer->id
  - Update: status = 'accepted', accepted_at = now()
  - Conversation: $bookingRequest->createConversation()
  - Redirection: back()->with('success')
  
- **reject(Request, BookingRequest)**: POST - Rejette demande
  - Auth: auth()->user()->tattooer
  - Validation: bookable_id === $tattooer->id
  - Update: status = 'rejected', rejection_reason
  - Redirection: back()->with('success')

  - Tattooer/Pierceur: $bookingRequest->bookable_id === $profile->id

- **accept(User, BookingRequest)**: bool
  - Tattooer/Pierceur: $bookingRequest->bookable_id === $profile->id
  - Admin: $user->isAdmin()
  
- **reject(User, BookingRequest)**: bool
  - Tattooer/Pierceur: $bookingRequest->bookable_id === $profile->id
